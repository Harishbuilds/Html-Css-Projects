StyleVista
A modern and elegant fashion showcase website built using HTML and CSS.
StyleVista presents curated fashion collections and trend inspirations through a clean layout and responsive design.

Project Overview :
StyleVista is a static fashion portfolio website.
It highlights curated fashion looks, trend inspirations, and modern styling concepts.
The website focuses on clean typography, visual storytelling, and responsive layout for different screen sizes.

Tech Stack :
HTML5
CSS3
Google Fonts

Features :
Responsive layout for desktop, tablet, and mobile screens
Fixed navigation bar for easy page access
Two separate pages for Work and Trends
Modern typography using Playfair Display and Pacifico fonts
Grid based image layout
Hover effects on images and navigation links
Clean and minimal design
Optimized spacing and alignment

Pages Included :
Home or Work page
Trends page

Each page has structured sections including navigation, headline quote, and image gallery layout.

Responsive Design :
The website adapts to different screen sizes using media queries.
Grid layout adjusts automatically for smaller screens.
Font sizes and spacing are optimized for mobile viewing.

Folder Structure :
index or StyleVista.html
trends.html
StyleVista.css
trends.css
img folder for images

How to Run the Project :
Download or clone the project folder
Open the folder in your system
Open StyleVista.html in your browser
No installation is required since it is a static website.
